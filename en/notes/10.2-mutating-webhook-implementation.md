```go
func (d *SimpleAppCustomDefaulter) Default(_ context.Context, obj *appsv1.SimpleApp) error {
    simpleapplog.Info("Defaulting for SimpleApp", "name", obj.GetName())

    // If image has no tag, append :latest
    if !strings.Contains(obj.Spec.Image, ":") {
        obj.Spec.Image = obj.Spec.Image + ":latest"
    }

    return nil
}
```

## How It Works

```text
User input: image: nginx       -> Webhook changes it to image: nginx:latest
User input: image: nginx:1.25  -> Tag already present, no change
```

## Understanding the Structure

Looking at the structure generated by the scaffold:

```go
// 1. Defaulter struct definition
type SimpleAppCustomDefaulter struct {}

// 2. Default method implementation
func (d *SimpleAppCustomDefaulter) Default(...) error { ... }

// 3. Registration with the Manager (SetupSimpleAppWebhookWithManager function)
ctrl.NewWebhookManagedBy(mgr, &appsv1.SimpleApp{}).
    WithDefaulter(&SimpleAppCustomDefaulter{}).
    Complete()
```

Comparison with the Controller's Reconcile pattern:

|              | Controller                              | Mutating Webhook                        |
| ------------ | --------------------------------------- | --------------------------------------- |
| Struct       | SimpleAppReconciler                     | SimpleAppCustomDefaulter                |
| Core function | Reconcile()                            | Default()                               |
| When called  | After CR change (after etcd storage)    | Before CR storage (before etcd storage) |
| Role         | Adjusts current state to desired state  | Auto-modifies values before storage     |
